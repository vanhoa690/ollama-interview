# 🧠 AI Quiz & Interview Readiness Platform

## 🎯 1. Mục tiêu sản phẩm

Xây dựng nền tảng giúp người học làm quiz và được AI phân tích năng lực,
định hướng học tập và phỏng vấn.

## 💡 2. Ý tưởng cốt lõi

Làm quiz → AI phân tích → đưa ra nhận xét + câu hỏi phỏng vấn.

## 🧱 3. Kiến trúc hệ thống

-   Frontend: React + Ant Design
-   Backend: Node.js (NestJS/Express)
-   Database: MongoDB
-   AI: Ollama local

## 🔄 4. Flow hoạt động

1.  User làm quiz
2.  Submit bài
3.  Backend chấm điểm
4.  Gửi dữ liệu sang AI
5.  AI phân tích
6.  Trả kết quả về frontend

## 🧠 5. Vai trò của AI

-   Phân tích kết quả
-   Đưa ra nhận xét
-   Gợi ý học tập
-   Sinh câu hỏi phỏng vấn

## 📊 6. Trải nghiệm người dùng

-   Trang quiz
-   Trang kết quả + AI analysis

## 🚀 7. Hướng phát triển

-   Adaptive quiz
-   Mock interview
-   Learning roadmap

## ⚙️ 8. Roadmap triển khai

-   Phase 4: AI nâng cao
